# 1이 될 때까지
- 주어진 값 a에 대해 최대한 많이 나눠야한다. -> a의 값을 줄일 때 2 이상의 수로 나누는 작업이 1을 빼는 작업보다 수를 더 많이 줄일 수 있기 때문
- 1.a가 b로 나누어 떨어진다면 나누고, 안된다면 1을 뺀다. 
- 1의 과정을 a가 1이 될 때까지 반복한다.  
## 작성한 코드
~~~python
a, b = map(int, input().split())
c = 0;

while ( a > 1) :
  if ( a % b == 0):
    a = a / b
  else :
    a = a - 1
  c = c + 1

print ( c )
~~~  
## 풀이
~~~python
a, b = map(int, input().split())
result = 0;

while True:
  target = (a // b) * b
  result += (a - target)
  a = target
  
  if ( a < b ) : break

  result += 1
  a //= b

result += (a - 1)
print (result)
~~~
