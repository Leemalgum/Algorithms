## 거스름 돈: 정당성 분석
### 가장 큰 화폐 단위부터 돈은 거슬러 주는 것이 최적의 해를 보장하는 이유는?
- 가지고 있는 동전 중 큰 단위가 항상 작은 단위의 배수 = 작은 단위의 동전들을 종합해 다른 해가 나올 수 없기 때문  
- 이처럼 최소한의 아이디어를 떠올리고 정당한지 검토해야함  
- 화폐의 종류=K라면 시간 복잡도=O(K), 동전 종류의 개수에만 영향을 받기 때문
